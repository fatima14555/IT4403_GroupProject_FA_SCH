# IT4403_GroupProject_FA_SCH
Overview
FilmCache is a fully functional Movie Discovery SPA that allows users to search for movies and TV shows, explore trending and top-rated content, view detailed information about films and actors, and manage a personal Favorites and Watchlist.

Features
Home — Trending daily and Top Rated movies
Discover — Search by title, filter by genre, sort by popularity/rating/date, paginated results
Movie Detail — Full info including backdrop, poster, genres, runtime, rating, overview, and cast
Actor Detail — Biography, birthdate, birthplace, and known filmography
My Lists — Favorites and Watchlist synced with TMDB account
Authentication — Full TMDB session login/logout flow

Tech Stack
jQuery — DOM manipulation, AJAX, event handling
TMDB API — All movie, TV, and user data
HTML / CSS — Single-file SPA structure
Google Fonts
GitHub Pages
